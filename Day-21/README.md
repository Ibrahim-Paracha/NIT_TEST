# Overview

In today's session, we learned how to efficiently create multiple files and directories using a single command. We also explored the difference between **absolute** and **relative** paths, along with the practical use of the **uptime** and **date** commands.

## Creating Multiple Files with a Single Command

Linux allows you to create multiple files at once using the `touch` command.

### Example

```bash
touch file-{01..03}
```

this creates 3 files ie file-01, file-02  and file-03.
